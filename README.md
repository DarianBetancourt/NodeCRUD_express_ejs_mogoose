# NodeCRUD_express_ejs_mogoose
CRUD for repositories
* download repository
* git clone https://github.com/DarianBetancourt/NodeCRUD_express_ejs_mogoose.git
* cd /NodeCRUD_express_ejs_mogoose
* npm install
* mongod
* npm start
